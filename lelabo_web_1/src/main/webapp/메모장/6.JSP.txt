[ JSP ]
	1. 정의 : Java Server Page 로 동적 페이지
	2. 역할 : HTML 과 JAVA 언어를 하나의 페이지 작성 가능.
		-> 확장자 : .jsp  , 
		-> MVC1패턴  
	3. JSP 비슷한 템플릿 
		1. JSP(java) 2.타임리프(spring) 3.react(페이스북)
	4. JSP 파일 HTTP 경로
		http://localhost:8080/tj2024b_web1/day05/index.jsp
		http://ip주소:port번호/프로젝트명/(wabapp생략)day05/index.jsp
	6. 문법
		1. <jsp:include page = "포함할파일경로.jsp" />  : 다른 jsp 파일을 동적으로 포함 
			-> HTML 언어는 html 안에 html 포함하는 기술 지원하지 않는다.
			-> 경로
				1. 파일명.jsp : 현재 jsp 와 불러올 jsp파일이 같은 폴더내 존재했을때.
				2. /패키지명/파일명.jsp : 현재 jsp 와 불러올 jsp파일이 다른 폴더내 존재했을때.
					-> 프로젝트내 (webapp폴더) 이하 경로부터 위치를 작성 해야 한다.

[ 경로 ]
	1. 컴파일(번역) 전 경로 / 개발자 코드를 작성한 곳 		( 개발자 PC 에 코드를 저장한 경로  )
		
	2. 컴파일(번역) 후 경로 / 서버에 배포 전 ( )			( 서버PC에 코드를 컴파일한 코드 경로 )
		<jsp:include page="/경로.jsp" />
		
	3. 컴파일(번역) 후 경로 / 서버에 배포 후 			( 클라이언트가 서버PC에 요청 경로 )
		http://localhost:8080/프로젝트명/
		<a href="/프로젝트명/경로" > </a>
		
	*  header.jsp 는 페이지 변경 될때 모든 페이지(공통) 에서 주로 사용하므로, 경로가 일정하지 않다.
		-> 즉] header 페이지 에서 경로 설정 할때는 상대경로 보다는 절대경로로 *고정 경로* 설정해야한다.
